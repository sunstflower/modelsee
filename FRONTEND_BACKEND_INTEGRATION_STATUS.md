## 🔧 部署说明

### 后端部署
```bash
cd backend
pip install -r requirements_layers.txt
pip install 'uvicorn[standard]' websockets
python -m uvicorn ml_backend:app --host 0.0.0.0 --port 8000
```

### 前端部署
```bash
npm install
npm install lucide-react
npm run dev
```
